# 📄 Past Paper Scraper

Made this out of sheer annoyance. I needed marking schemes and question papers downloaded in bulk for printing
as well as ingesting into NotebookLM. The script is very rough around the edges because Cambridge in their
infinite wisdom has kept on changing component codes, subject codes and other stuff over the years. I've tried to
make it one size fits all as best as I can. I'll maybe improve it if I have time.

If you want to add new subjects or expand the program, feel free to make a fork. I just thought I would
upload it here hopefully to help someone out. It's pretty limited as is right now because I've kept it personalised
to my needs.

I want to credit [BestExamHelp](https://bestexamhelp.com/) since most of the scraping is done on their side.
I added a random sleep time in between individual requests so their servers aren't overloaded.
